# v0.3.0 Release Notes (2026-05-08)

## 版本口径说明

- 该版本按 `docs/VERSION_REBASELINE_PLAN.md` 对外口径发布。
- 公开版本从 `v0.2.1` 连续到 `v0.3.0`，历史 `v0.4.5~v0.6.0` 视为内部演进标签。

## 用户可见增强

- 新增 `evolution_report`：输出执行来源、错误分类、未修复失败和建议。
- 路由前自治策略：`AUTONOMY_MODE=off|assist|auto`。
- `text_wordcount` 输入回退兼容（`context.text` 缺失时自动取 `task/input`）。

## 示例调用

```bash
aion-cli --json "evolution report"
```

返回应包含：
- `summary`
- `sources`
- `errors`
- `recommendations`
- `unresolved_failures`

## 质量指标（对比 v0.2.1）

- success_rate（10 次 YAML 解析基准）：`0% -> 100%`
- P95 latency（同基准）：`9727ms -> 13291ms`

## 验证记录

- `cargo test -p aion-router --quiet`：71 passed
- `docs/forge-evidence.md`：包含可复现实操验收命令
